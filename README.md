# Week 3 – Machine Learning Prediction System using Scikit-learn

## Project Overview

This project was developed as part of the Week 3 Internship Task on Machine Learning Fundamentals, Model Training, and Prediction Systems using Scikit-learn.

The objective of this project is to understand the complete Machine Learning workflow including data preprocessing, model training, evaluation, prediction generation, and deployment using Streamlit.

---

## Dataset

Dataset Used: Diabetes Prediction Dataset

The dataset contains information about patients including:

* Gender
* Age
* Hypertension
* Heart Disease
* Smoking History
* BMI
* HbA1c Level
* Blood Glucose Level

Target Variable:

* Diabetes (0 = No Diabetes, 1 = Diabetes)

Dataset Source:

https://www.kaggle.com/datasets/iammustafatz/diabetes-prediction-dataset

## Data Preprocessing

The following preprocessing steps were performed:

1. Dataset loading
2. Missing value checking
3. Duplicate record removal
4. Categorical data encoding
5. Feature selection
6. Data normalization using StandardScaler
7. Train-Test Split

---

## Machine Learning Models

### Logistic Regression

Used for binary classification to predict whether a patient has diabetes or not.

### Random Forest Classifier

Used to improve prediction performance using an ensemble learning approach.

---

## Model Evaluation

The models were evaluated using:

* Accuracy Score
* Precision Score
* Recall Score
* Confusion Matrix

Visualization techniques:

* Confusion Matrix Heatmap
* Diabetes Distribution Chart
* Model Performance Comparison

---

## Prediction System

A prediction system was developed where users can enter patient information and receive instant diabetes prediction results.

Possible Outputs:

* No Diabetes
* Diabetes Detected

---

## Streamlit Application

The Streamlit application includes:

* Dataset Overview
* Model Information
* User Input Fields
* Prediction Button
* Prediction Results
* Performance Visualization

Run the application using:

python -m streamlit run app.py
